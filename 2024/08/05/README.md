    - 해외 사업에서는 매출이 전년 대비 5.6% 증가했으며, 원재료 투입 단가 하락과 생산성 향상으로 영업이익이 전년 대비 37.6% 증가했습니다. 인도에서는 초코파이 제3라인 가동 효과로 건과 매출이 성장했고, 인도 빙과도 우호적인 날씨 덕분에 성장을 보였습니다. 러시아에서는 루블화 부담이 줄어들면서 매출이 증가했고, 카자흐스탄에서는 환율 변동으로 인해 부진했으나 3~4분기에는 매출 회복이 예상됩니다.
