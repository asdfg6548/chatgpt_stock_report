# 2024.05.29 stock report
## 리가켐바이오
- 미래에셋증권에서 리가켐바이오에 대해 투자의견 '매수'를 제시하며 목표주가를 86,000원으로 설정했습니다. 이는 상승여력 35.6%를 의미합니다. 목표주가 산정에는 2024~2030년 추정치를 반영한 DCF 모델을 사용했으며, 2030년까지 3개의 신약 출시를 기대하고 있습니다. 영구성장률은 1%, 가중평균자본비용(WACC)은 8.2%로 적용되었습니다. 현재 리가켐바이오의 P/E 비율은 25년도 기준으로 14.6배에 달하며, 이는 씨젠, 이뮤노젠, 다이이찌산쿄와 같은 타사와 비교해 매우 유리한 밸류에이션입니다.
