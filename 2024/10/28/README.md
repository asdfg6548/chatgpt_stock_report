# 2024.10.28 stock report
## 종근당
- 3분기 2024년 실적은 원가율 상승으로 기대에 미치지 못함. 매출액은 4,085억 원으로 전년 대비 3.1% 증가했으나, 영업이익은 252억 원으로 52.5% 감소함. 기존 품목인 프롤리아와 아토젯의 매출 성장과 신규 도입 품목인 펙수클루, 고덱스의 매출 덕분에 전체 매출은 증가했으나, 도입 상품 중심의 매출 성장과 글리아티린의 충당금 반영으로 원가율은 69%로 상승했음. 해당 충당금은 콜린 알포세레이트 제제의 임상 재평가와 건강 보험 지급액 환수 계약과 관련이 있음. 목표가는 148,000원이며, 투자의견은 매수임.
