# 2024.08.28 stock report

## KT
- 추천 매수 의견과 목표 주가 5만원을 유지하며, 아직 주가가 충분히 오르지 않았다고 평가된다. 2024년 9월 통신서비스 투자 전략에서도 KT의 매수 의견을 지지한다. 주요 이유로는 2024년 연간 연결 영업이익이 예상보다 나쁘지 않을 전망, 부동산 자회사의 아파트 분양 일정에 따른 대규모 이익 발생 예측, 연간 약 7천억 원 수준의 주주이익환원 규모 유지, 그리고 2천억 원 규모의 자사주 소각을 통해 주주이익환원 규모가 40% 증가한 데 비해 주가는 그만큼 오르지 못했다는 점이 꼽힌다.
## LG유플러스

## 뷰웍스
